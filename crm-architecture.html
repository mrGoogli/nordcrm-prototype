05

Security & Compliance

🔐
Identity & Access
Azure Entra ID with MFA, Conditional Access policies, RBAC per CRM module, and PIM for privileged roles.
🔒
Data Encryption
TLS 1.3 in transit, AES-256 at rest. Azure Key Vault for secrets and certificates management.
🇪🇺
GDPR Compliance
Data residency in EU regions, right-to-erasure workflows, consent tracking, and audit logs for all PII access.
🏦
Financial Data (Stacc)
PSD2 / Open Banking compliant data handling. Stacc data isolated per tenant, with field-level encryption on financial records.
🕵️
Audit & Monitoring
Full audit trail in Azure Monitor. Microsoft Sentinel SIEM integration. Real-time anomaly detection and alerting.
🏢
Multi-tenancy
Tenant isolation at data and API level. Each org gets isolated Cosmos DB container with dedicated throughput.
